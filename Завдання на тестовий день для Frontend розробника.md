**Завдання на тестовий день для Frontend розробника**

*За основу завдання взято Mini Request System – зроблене вами попереднє ТЗ.*  
*ТЗ складається з двох етапів:* 

- *Перший етап – реалізація функціоналу відповідно до наведених нижче пунктів.*   
- *Другий етап – демонстрація екрана та коротка live coding сесія.*  
  ***Час виконання першого етапу**: до 4-ох годин.*

    
  ***Перший етап***  
    
1. Видалення запитів:  
   1. для кожного запиту кнопка **«Видалити»;**  
   2. запит повністю видаляється із системи;  
   3. має бути підтвердження дії (наприклад, модальне вікно).  
2. Зміна теми:  
   1.  додати можливість перемикання теми інтерфейсу на **Темну** / **Світлу.**  
3. Локалізація мови:  
   1. додати перемикач мови;  
   2. підтримувані мови: **Українська** та **Англійська;**  
   3. дефолтна мова має вибиратися по хедеру, якщо попередніх мов немає у хедері юзера, дефолтною має вибратися **англійська**.  
4. Логування дій:  
   1. для **Менеджера** додати вкладку **Логи;**  
   2. у логи записувати роль, час і назву дії;  
   3. у логах мають бути усі дії зроблені як **Юзером**, так і **Менеджером**.
